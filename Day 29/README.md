# Day 29 – Password Manager 🔐

This project is part of **Dr. Angela Yu’s 100 Days of Python** course, Day 29.  
The objective of this challenge is to build a **Password Manager GUI application** using **Tkinter**, focusing on user input handling, file I/O, and GUI design.

---

## 📌 Project Overview

This Password Manager allows users to:
- Generate secure random passwords
- Save website login credentials locally
- Retrieve and manage stored credentials
- Interact with the application through a clean Tkinter-based GUI

The project reinforces event-driven programming and practical GUI development in Python.

---

## 🛠️ Features

- **Password Generator**
  - Creates strong, randomized passwords
- **Save Credentials**
  - Stores website, email/username, and password
- **Input Validation**
  - Prevents saving empty fields
- **Clipboard Support**
  - Automatically copies generated passwords
- **Simple & Clean GUI**
  - Built using Tkinter widgets

---

## 🎮 How It Works

1. Enter the **website name**
2. Enter your **email/username**
3. Generate or enter a **password**
4. Click **Add** to save credentials

Saved credentials are stored locally in a file for later reference.
