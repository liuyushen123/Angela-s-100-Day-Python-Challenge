The goal of this challenge is to code every single day, progressively moving from foundational programming concepts to advanced software development, automation, and web frameworks.

As a computer science student, I am using this challenge to bridge the gap between academic theory and practical, real-world application. While my university coursework provides a strong foundation, this bootcamp serves as a hands-on sandbox. My specific goals for this repository include:

## 🏗️ Course Curriculum & Repository Structure

The repository is organized chronologically by day. The bootcamp is divided into four major skill-level phases:

### Phase 1: Beginner (Days 1 - 14)
Focuses on the core syntax and fundamentals of Python.
* **Concepts:** Variables, Data Types, Control Flow (If/Else), Randomization, Loops, Functions, and Dictionaries.
* **Key Projects:** Band Name Generator, Tip Calculator, Hangman, Caesar Cipher, Secret Auction, and a text-based Blackjack game.

### Phase 2: Intermediate (Days 15 - 31)
Introduces Object-Oriented Programming and graphical interfaces.
* **Concepts:** OOP, Classes/Objects, The Turtle Module, Tkinter (GUI), Event Listeners, State Management, List/Dictionary Comprehensions, and Error Handling.
* **Key Projects:** Coffee Machine, Quiz API App, Pomodoro GUI Timer, Password Manager, and classic arcade game clones (Snake, Pong, Turtle Crossing).

### Phase 3: Intermediate+ (Days 32 - 58)
Dives into the web, working with external data, and automation.
* **Concepts:** API Endpoints, Authentication, Web Scraping (BeautifulSoup), Browser Automation (Selenium), Sending Emails (SMTP), and Data Analysis (Pandas).
* **Key Projects:** Automated Birthday Emailer, ISS Overhead Notifier, Rain Alert SMS system, Flight Deal Finder, and an Automated Spotify Playlist Creator.

### Phase 4: Advanced (Days 59 - 100)
Transitions into advanced web development and database management.
* **Concepts:** Web Dev Foundations (HTML/CSS/Bootstrap), Flask Framework, Jinja Templating, SQLite, SQLAlchemy, REST API creation, and Deployment.
* **Key Projects:** Personal Blog site, RESTful Cafe API, E-commerce storefront, and custom web applications.
